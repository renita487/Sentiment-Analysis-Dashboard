# 💬 Sentiment Analysis Dashboard

An interactive AI-powered web application that analyzes user input and classifies it into Positive, Negative, or Neutral sentiment.

## 🚀 Features
- Real-time sentiment prediction
- Interactive web interface using Streamlit
- Supports Positive, Negative, and Neutral classification
- Live sentiment tracking with a dynamic bar chart

## 🛠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas

## 🧠 How it works
- Text is converted into numerical data using CountVectorizer
- A Logistic Regression model is trained on sample data
- The model predicts sentiment based on user input
- Results are displayed instantly with visual feedback

## 📊 Output
Users can input any sentence and view:
- Sentiment result (Positive/Negative/Neutral)
- A real-time dashboard showing sentiment distribution
